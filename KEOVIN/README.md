# KEOVIN Web

Static product website for KEOVIN / Torn-AgentPay.

## Files

- `index.html` - main public entry.
- `code.html` - editable working copy, currently synchronized with `index.html`.
- `assets/` - logo assets used by the page.
- `_previews/` - generated browser screenshots for review.

## What The Page Covers

- AI-facing payment protocol positioning.
- Product runtime: buyer, seller, Tron payment channel, MCP tools, skill runner.
- Install paths: skill-only, generic MCP, Codex-style skill, and external host templates.
- Interactive protocol cockpit, dashboard, docs, and contact form feedback.
- Security notes: replay-safe channels, protected ops, onboarding hardening, release safety.
- Current product copy source: `PRODUCT_COPY.md`.

Open `index.html` directly in a browser.
