# Sample Project

This folder is a minimal project-shaped template showing how an integrator could wire:

- a seller-side merchant app
- a buyer-side agent runtime

Files:

- `merchant_service.py`: sample seller app bootstrap
- `agent_client.py`: sample buyer runtime bootstrap

Suggested usage:

1. copy `../examples/.env.example` to `.env.local`
2. adjust addresses, keys, and repository paths
3. run the merchant service
4. run the agent client
